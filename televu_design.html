Design

Considerations - 3 points

We have considered the following three points while we were coming up with new design for the client.

  • convenient based on frequent use
    • the features that are most frequently used should be well-spotted
  • ease on navigation
    • it is important to remind the customers about what they can do as the next step
  • interactivity
    • the product needs to be interactive. it should provide feedback after each action

    Branding & Style

    The brand tone for TeleVU Incorporation is agile, but it was clean and neat.
    Therefore, we have been consistent with blue colour, which was the primary colour for the design, but we have also mixed with grey, red, and green as supplementary colours.

    Visual Drafts

    We all drew initial sketch for every page of the web application individually based on the user flow to get a better sense of how they should look and feel. After then, we have merged the ideas based on the client's choice:

    Changes in History menu

  • taking an input from the client, we have added the categories to make it more organized
  • the client's customers wanted to reduce the time used to spent in looking for a particular file, so we have added a search engine called "filter & search"

    • Changes in Video-Conference session window

  • the client did not like the empty spaces being disproportionate compared to the size of icon layouts and other screens for the participants
  • the client asked us to utilize the empty white spaces and readjust the size of screen in the centre as enlarged as possible
  • the client wanted a clear difference between the icons for the web application and the icons for the mobile remote-control; therefore, we have tried to make them as intuitive as possible while keeping the clear difference.

    • Changes in GlassVU web page

  • we realized that the product is not interactive with the customers.
    i.e. they find a designated person to call, but they do not understand when they can call the person
    --- sometimes the call buttons are white not clickable, sometimes the buttons are green and clickable
